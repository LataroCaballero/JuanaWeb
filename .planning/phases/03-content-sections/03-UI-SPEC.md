## Interaction Contracts

| Component | Interaction | Mechanism | Duration |
|-----------|-------------|-----------|----------|
| Nav | Glassmorphism on scroll > 80px | JS scroll listener, toggle `.scrolled` class | instant class add/remove |
| Nav links | Hover background fill | `hover:bg-electric hover:text-white` | `transition-none` â€” instant flash, no ease |
| Location cards | Grayscale â†’ color | `filter: grayscale(1)` â†’ `grayscale(0)` on `group-hover` | `transition-all duration-700` (700ms) |
| Marquee | Continuous scroll | CSS `@keyframes marquee` | 20s linear infinite |
| Marquee | Pause on reduced motion | `@media (prefers-reduced-motion: reduce)` CSS | n/a â€” CSS only |
| Footer IG link | Hover color | `hover:text-electric` | `transition-none` â€” instant |
| All CTA buttons | Hover invert | `hover:bg-white hover:text-electric` | `transition-none` â€” instant |

The "no slow transitions" rule from DESIGN.md applies to all interactive elements except the location card grayscale reveal (which is intentionally cinematic at 700ms).
