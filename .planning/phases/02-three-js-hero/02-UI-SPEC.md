## Spacing Scale

Declared values (all multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |

| xl | 32px | Layout column gap |
| 2xl | 48px | Hero vertical padding mobile |
| 3xl | 64px | Hero section min-height padding desktop |

Exceptions:
- Canvas overlap offset desktop: `margin-left: -48px` (negative, breaks grid intentionally — Organic Brutalism). Source: CONTEXT.md D-08.

- `body::after` grain overlay: z-index 9999, `pointer-events: none` — canvas must use z-index < 9999 to receive grain texture on top.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#131313` | Hero background, page surface — `var(--color-surface)` |
| Secondary (30%) | `#1a1a1a` / `#353534` | Depth layering if needed — `var(--color-surface-low)` / `var(--color-surface-high)` |
| Accent (10%) | `#0055ff` | 3D logo material color only — `var(--color-electric)` |
| Destructive | `#ffb4ab` | Not applicable in this phase — `var(--color-error)` declared but unused |

Accent reserved for: **3D logo material** (`MeshStandardMaterial color: #0055ff`) only. No other element in the hero uses Electric Blue in this phase. The h1 text is `var(--color-on-surface)` (`#e5e2e1`), not Electric Blue.

Source: CONTEXT.md D-03, global.css @theme tokens.

### Three.js Material Spec

```
MeshStandardMaterial:
  color:     #0055ff   (var --color-electric)
  metalness: 0.7
  roughness: 0.2
```

### Lighting Spec (Claude's Discretion — resolved here)

```
AmbientLight:
  color:     #ffffff
  intensity: 0.4

DirectionalLight:
  color:     #b6c4ff   (electric-light — warm metallic reflection)
  intensity: 1.2
  position:  [5, 8, 5]   (top-right front — activates metallic specular on upper face)
```

Rationale: AmbientLight at 0.4 keeps dark areas visible without washing out the metallic effect. DirectionalLight using electric-light (`#b6c4ff`) at top-right creates a premium chrome-blue sheen consistent with the brand signature. No fill light — deep shadows on the left side reinforce the brutalist edge.

## Geometry & Animation Spec

This section is unique to this phase — Three.js visual contracts that the executor treats as locked.

### SVG Extrusion

| Property | Value | Source |
|----------|-------|--------|
| SVG file | `src/assets/smiley-logo.svg` | CONTEXT.md D-01 |
| SVG structure | `<path>` elements only — no `<use>`, no `<symbol>` | CONTEXT.md D-01 |
| SVG contents | Outer circle + two arrow-eyes | CONTEXT.md D-01 |
| Extrusion depth | 10 units (Three.js scene units) | CONTEXT.md D-05 range: 8–12 |
| Loader | `THREE.SVGLoader` + `THREE.ExtrudeGeometry` | CONTEXT.md D-01 |

### Canvas Dimensions

| Property | Value | Source |
|----------|-------|--------|
| Desktop canvas size | 45vmin × 45vmin | Claude's Discretion — fills half the viewport height without overflowing |
| Mobile canvas size | 70vw × 70vw | Claude's Discretion — dominant on mobile stack |
| Aspect ratio (CSS) | `aspect-ratio: 1 / 1` | CONTEXT.md D-13 — CLS prevention |
| Background | transparent — `renderer.setClearColor(0x000000, 0)` | Inherits `#131313` from page |
| z-index | 100 (below grain overlay at 9999) | CONTEXT.md code_context — grain must cover canvas |

### Idle Animation

| Property | Value | Source |
|----------|-------|--------|
| Y-axis rotation | `mesh.rotation.y += 0.003` rad/frame | CONTEXT.md D-06 |
| Float amplitude | 0.05 units | CONTEXT.md D-06 |
| Float frequency | `Math.sin(elapsed * 0.8)` — slow, deliberate | Claude's Discretion within D-06 style |
| Entry animation | None — mesh appears and floats immediately | CONTEXT.md D-07 |
| prefers-reduced-motion | Float and rotation paused — `window.matchMedia('(prefers-reduced-motion: reduce)')` | CONTEXT.md D-06 |

### Hero Layout

| Breakpoint | Layout | Overlap |
|------------|--------|---------|
| mobile (< 768px) | Stack vertical: h1 top, canvas bottom | `margin-top: -24px` on canvas — logo rises 24px into h1 space |
| desktop (≥ 768px) | CSS Grid 2-col: h1 left, canvas right | `margin-left: -48px` on canvas — logo invades 48px into h1 column |

Source: CONTEXT.md D-08, D-09. Exact pixel values are Claude's Discretion resolved here. Both values chosen to be "un poquito" — visible but not aggressive.

## Interaction States

| State | Behavior |
|-------|----------|
| Before viewport entry | Canvas not initialized — IntersectionObserver pending. Reserved space visible via `aspect-ratio` CSS. No JS loaded. |
| Canvas entering viewport | `IntersectionObserver` callback fires. `import('./hero-canvas.js')` loads Three.js dynamically. Renderer starts. |
| Canvas active | Idle animation runs: float + slow Y-rotation. |
| prefers-reduced-motion active | Animation loop stops after first render. Mesh visible but static. |
| Page unload / component teardown | `geometry.dispose()`, `material.dispose()`, `renderer.dispose()`, `renderer.forceContextLoss()` — no GPU leak. |

Source: CONTEXT.md D-12, D-13, D-14, REQUIREMENTS.md HERO-04.
