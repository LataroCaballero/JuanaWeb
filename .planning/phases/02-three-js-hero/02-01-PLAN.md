    if (wrapper) {
      const observer = new IntersectionObserver(

    **Part A: Create `src/scripts/hero-canvas.ts`**

    Export a single async function `initHeroCanvas(wrapper: HTMLElement): Promise<() => void>`.

    The function must implement these steps in order:

    1. **Scene setup:**
       - `new THREE.Scene()`
       - `new THREE.PerspectiveCamera(45, 1, 0.1, 1000)` — aspect ratio 1 because canvas is square
       - `new THREE.WebGLRenderer({ antialias: true, alpha: true })`
       - `renderer.setClearColor(0x000000, 0)` — transparent, inherits #131313 from CSS
       - `renderer.setSize(wrapper.clientWidth, wrapper.clientWidth)` — square
       - `renderer.setPixelRatio(Math.min(window.devicePixelRatio, 2))` — cap at 2x for perf
       - `wrapper.appendChild(renderer.domElement)`

    2. **Lighting (per UI-SPEC):**
       - `new THREE.AmbientLight(0xffffff, 0.4)`
       - `new THREE.DirectionalLight(0xb6c4ff, 1.2)` at position `(5, 8, 5)`

    3. **Load SVG and build mesh:**
       - `import { SVGLoader } from 'three/addons/loaders/SVGLoader.js'`
       - `const data = await loader.loadAsync('/assets/smiley-logo.svg')`
       - Create a `THREE.Group()`
       - `group.scale.y *= -1` — fix SVG Y-axis inversion (CRITICAL per RESEARCH.md Pitfall 2)
       - For each `path` in `data.paths`:
         - `SVGLoader.createShapes(path)` (NOT `path.toShapes()` — per RESEARCH.md State of the Art)
         - For each shape: `new THREE.ExtrudeGeometry(shape, { depth: 10, bevelEnabled: false })`
         - Material per path: if path fill is `#131313` (eye), use `new THREE.MeshStandardMaterial({ color: 0x131313, metalness: 0.5, roughness: 0.4 })`. If fill is `#0055ff` (circle), use `new THREE.MeshStandardMaterial({ color: 0x0055ff, metalness: 0.7, roughness: 0.2 })` (D-03).
         - Add mesh to group
       - Position eye meshes slightly forward: `mesh.position.z = 1` on eye paths (so eyes are visible on front face of extruded circle)

    4. **Center and scale:**
       - `const box = new THREE.Box3().setFromObject(group)`
       - `const center = new THREE.Vector3(); box.getCenter(center)`
       - `group.position.sub(center)` — center in scene
       - Compute auto-fit scale: `const size = box.getSize(new THREE.Vector3()); const maxDim = Math.max(size.x, size.y, size.z); const fitScale = 3.0 / maxDim;` then `group.scale.multiplyScalar(fitScale)` — fits the logo to fill roughly 3 units, visible at camera z=5 with FOV 45
       - Set `camera.position.z = 5`

    5. **Animation loop (D-06):**
       - Check `window.matchMedia('(prefers-reduced-motion: reduce)').matches`
       - If NOT reduced motion: start RAF loop with:
         - `group.rotation.y += 0.003` (slow Y rotation)
         - `group.position.y = Math.sin(elapsed * 0.8) * 0.05` (float, per UI-SPEC: 0.8 rad/s, 0.05 amplitude)
         - `renderer.render(scene, camera)`
       - If reduced motion: render ONE static frame only, do NOT start RAF loop

    6. **Teardown function (HERO-04, D-14):**
       - `cancelAnimationFrame(rafId)`
       - `group.traverse((obj) => { if (obj.isMesh) { obj.geometry.dispose(); obj.material.dispose(); } })`
       - `renderer.forceContextLoss()` — BEFORE renderer.dispose() (per RESEARCH.md Pitfall 5 / GitHub #27100)
       - `renderer.dispose()`
       - Bind to `window.addEventListener('pagehide', teardown, { once: true })`
       - Return the teardown function from `initHeroCanvas` for explicit cleanup

    Import paths — use `three/addons/` (NOT `three/examples/jsm/` — per RESEARCH.md State of the Art):
    ```typescript
    import * as THREE from 'three';
    import { SVGLoader } from 'three/addons/loaders/SVGLoader.js';
    ```
